# Rekommendationskomponent

Denna sida beskriver den övergripande arkitekturen för "Rekommendationer" inom mCommerce-plattformen

## Arkitektur

Lösningen består av ett antal komponenter.
<img src={arch_imagefile} />

### Frontend
* **RecommendationComponent** [Presentationskomponent](?path=/story/experiment-recommendation--kartskikt)
som listar rekommendationer för den inloggade användaren och låter denne interagera med rekommendationerna.
* **RecommendationService** kommunicerar med backend-tjänsterna.

### Backend
* **Sökmotor** som skapar rekommendationer baserad på inloggad användares kundtillhörighet.
Se avsnittet AI nedan.
* **Produktkatalog** innehåller metdata om de olika produkterna. Från denna hämtas bilder och beskrivningar om de
olika produkterna
