Introfilm

Sökpanelen

Adressök

För att söka på adress kan man först söka på gatuadress och ort, som t ex Stationsgatan 7, Luleå. För att få träff förslag behöver du skriva tre tecken så att du sedan kan gå vidare med det du sökte.

Fastighetssök

För att söka på fastighetsbeteckningen söker man först på kommun och sedan får man träff på fastighetsbeteckningen t.ex vid denna träff; Skellefteå Abborren 1. För att komma till rätt träff scrollar du ner till den träffen och trycker enter. Alternativt vänsterklickar du på den träffen du vill komma till.

Lagerpanelen

Lager

När du hovrar över en specifik parameter så får du en liten beskrivning om den specifika parametern. Om du klickar i en parameter så väljs den och visas på kartan. Vid vissa parametrar finns det flera nivåer som går att välja som t.ex. Nationalpark. Om du vänsterklickar på kugghjulet kan du ställa in transparens.

Bakgrundskartor

I denna funktion kan man välja vilken bakgrundskarta som ska användas i applikationen:

- Bakgrundskarta med fastighetsgränser (1:10 000)

- Bakgrundskarata som övergår till skuggad terrängmodell med fastighetsgränser (1:10 000)

- Ortofoto med fastighetsgränser (1:10 000)

Informationspanel

Valda objekt listas i Informationspanelen till vänster.

Zooma till aktuellt objekt genom att klicka på zoom_in till vänster om respektive objekt.

Fastigheter

För mer information om fastigheter klicka på fastighetsbeteckningen i listan. Du kommer då vidare till Metria FastighetSök där du hittar mer information om vald fastighet. Observera att särskilt konto behövs för applikationen. Tillfälliga inställningar kan göras med kugghjulet settings.

Identifiera fastighet

Det går även att klicka på valfri plats på kartan för att få fram det fastighetsdelområde som du efterfrågar.

Skoglig analys - panelen

Skogliga grunddata

En beräkning utförd för valt fastighetdelområde baserad på skoglig grunddata från Skogsstyrelsen. Resultatet presenteras i analyspanelen.

Trädslagsfördelning

En beräkning utförd för valt fastighetdelområde baserad på nationellt marktäckedata från Naturvårdsverket. Resultatet presenteras i analyspanelen.

Avverkningsstatistik

En beräkning utförd för valt fastighetdelområde baserad på utförda avverkningar från Skogsstyrelsen. Resultatet presenteras i analyspanelen.

Huggningsklasser

En beräkning utförd för valt fastighetdelområde baserad på trädhöjdsraster från Metria. Resultatet presenteras i analyspanelen.

Skyddade områden

En beräkning utförd för valt fastighetdelområde baserad på Naturvårdsverket, Riksantikvarieämbetet och Skogsstyrelsens skyddade områden/områden med naturvärde. Resultatet presenteras i analyspanelen.

Verktygspanelen

Det finns fyra val i tjänstens nedre mitt som du kan använda dig av för att selektera din analys.

Välj verktygsinställningar

För att endast markera ett fastighetsdelområde klicka på ” Verktygsinställningar” och bocka ur kryssrutan. Klickar du på en specifik fastighet kommer samtliga fastighetsdelområden tillhörande den fastigheten att markeras och beräknas. I fastighetspanelen under fastighetsbeteckningen finner du antalet fastighetsdelområden tillhörande fastigheten.

Välj objekt

Klicka på ”välj objekt” för att kunna välja en fastighet.

Välj fastighetsdelområde

Klickar du på en specifik fastighet kommer samtliga fastighetsdelområden tillhörande den fastigheten att markeras och beräknas. I fastighetspanelen under fastighetsbeteckningen finner du antalet fastighetsdelområden tillhörande fastigheten.

Rita eget område

Genom att rita en polygon kan du själv avgränsa ett område som du vill göra en beräkning på. Håll nere tangenten shift för att frihandsrita.

Mät område

Klicka på ”Mät område” för att mäta en yta över ett specifikt område. Håll nere tangenten shift för att frihandsrita.

Mät avstånd

Klicka på ”Mät avstånd” för att mäta avstånd över ett specifikt område. Håll nere tangenten shift för att frihandsrita.

Markera flera fastigheter

Håll nere ”ctrl” och klicka på de fastigheter som du vill markera. Vill du endast markera flera fastighetsdelområden klicka på verktygsinställningar och bocka ur välj hela fastigheten. Håll inne ”ctrl” och klicka på de fastighetsdelområden som du vill markera.
