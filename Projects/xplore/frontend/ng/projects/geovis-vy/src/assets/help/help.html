Sökpanelensearch

Adressök

För att söka på adress kan man först söka på gatuadress och ort, som t ex Stationsgatan 7, Luleå. För att få träff förslag behöver du skriva tre tecken så att du sedan kan gå vidare med det du sökte.

Fastighetssök

För att söka på fastighetsbeteckningen söker man först på kommun och sedan får man träff på fastighetsbeteckningen tex vid denna träff; Skellefteå Abborren 1. För att komma till rätt träff scrollar du ner till den träffen och trycker enter. Alternativt vänsterklickar du på den träffen du vill komma till.

Lagerpanelen

Lagerpanelen får man fram genom att klicka på layers uppe till höger.

Lager

Grupplagren kan fällas ut genom att klicka på pilen till vänster om namnet. Du kan tända och släcka olika lager genom att bocka i och ur kryssrutorna för varje lager. Med opacity bredvid respektive lager kan man ställa in transparens.

I kartan kan man klicka på ett objekt för mer information. Informationen visas i informationsrutan till vänster.

Bakgrundskartor

Olika bakgrundskartor kan användas i applikationen och de kan väljas antingen från lagerlistan eller genom att klicka direkt på miniatyrkartona nere till vänster i bild (den funktionen är inte tillgänglig i mobilläge).

Mätverktyg

Mätverktyg område format_shapes

Ett verktyg för att mäta arean på godtyckligt område. Det går att rita ett område genom att antingen klicka ut punkter på kartan, eller att frihandsrita genom att hålla nere Skifttangenten, klicka och dra. Avsluta ritning genom att dubbelklicka.

Verktyget har även stöd för att i efterhand modifiera utritade polygoner/områden. För att modifiera, ta tag i den ritade polygonen med muspekaren och dra.

Mätverktyg avstånd timeline

Ett verktyg för att mäta sträckan mellan två eller fler punkter. Det går att rita en linje genom att antingen klicka ut punkter på kartan, eller att frihandsrita genom att hålla nere Skifttangenten, klicka och dra. Avsluta ritning genom att dubbelklicka.

Verktyget har även stöd för att i efterhand modifiera utritade linjer. För att modifiera, ta tag i den ritade linjen med muspekaren och dra.

Platsinformation info

Genom att klicka någonstans i kartan visas platsinformation i form av koordinat.

Informationspanel

Valda objekt listas i Informationspanelen till vänster.
Zooma till aktuellt objekt genom att klicka på zoom_in till vänster om respektive objekt.

Fastigheter

För mer information om fastigheter klicka på fastighetsbeteckningen i listan. Du kommer då vidare till Metria FastighetSök. Observera att särskilt konto behövs för applikationen.
Export av valda fastigheter görs med knappen save_alt.
Tillfälliga inställningar kan göras med kugghjulet settings.

Verktyg

Verktygsknapparna till vänster underlättar navigering i kartan. Man kan zooma med plus add och minustecknet remove. Man kan navigera tillbaka till startvyn med home. Under förutsättning att din GPS-funktion är aktiverad på din enhet, kan du hitta din position i kartan genom att klicka på my_location.

Precisionen för ”Min position” kan variera beroende på vilken webbläsare som används, vi rekommenderar att någon av följande webbläsare används Microsoft Edge, Firefox, eller Google Chrome.

För bästa möjliga precision rekommenderar vi att du använder en mobilenhet, då du generellt sett får än mer precis position. Dock kan även geografiska förutsättningar och enhetens specifika inställningar ha inverkan på positioneringen, vilket kan leda till viss avvikelse från din faktiska position.

Urval

Fastighetsurval kan ske antingen genom att klicka på enskilda fastigheter eller selektera fler fastigheter genom att samtidigt hålla ner ”Ctrl”-knappen. Urval kan också ske genom att dra en godtycklig polygon där 2 varianter är möjliga för urval; ”Välj det som berör en polygon” alternativt ”Välj det som ligger helt innanför en polygon”. Ett sista alternativ är att göra urval via en godtycklig linje, "Välj det som berör en linje".
